Windows Usage

Advanced MediaTek recovery workflow for Windows systems.

Driver Installation (MANDATORY)

Before running any Penumbra script on Windows, you must install the USB driver. 

Run: PenumbraDrivers.exe
  • Select libusb
  • Click Install
  • Reboot recommended

If this step is skipped, antumbra will remain stuck at: 

Waiting for MTK device‚Ä¶

IMEI Backup (RECOMMENDED FIRST)

imei-backup.bat

This backs up critical radio and identity partitions. Always do this before flashing firmware.

Full Unbrick / Flash

unbrickscript.bat
  • Flashes all required A/B firmware partitions
  • Flashes super.img last
  • Does NOT flash preloader
