Windows Usage Guide

Instructions for using the Penumbra Toolkit on Windows.

1. Install USB Driver (MANDATORY)

Before running any script, install the Penumbra USB driver.

Run: PenumbraDrivers.exe
  • Select libusb
  • Click Install
  • Reboot recommended

If skipped: antumbra will show Waiting for MTK device…

2. Folder Structure (DO NOT CHANGE)

Penumbra\
├── bin\
│   ├── antumbra.exe
│   └── da.bin
├── FIRMWARE\
│   └── *.img
├── imei-backup.bat
├── imei-restore.bat
├── unbrickscript.bat
└── PenumbraDrivers.exe

3. IMEI Backup (RECOMMENDED FIRST)

imei-backup.bat

Backs up IMEI and radio calibration data.

4. Full Unbrick / Flash

unbrickscript.bat
  • Flashes all A/B partitions
  • Flashes super.img last
  • No preloader flashing

5. IMEI Restore (Optional)

imei-restore.bat

Restore only from the same device.
