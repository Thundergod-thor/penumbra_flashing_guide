# Penumbra Toolkit

A cross-platform **MediaTek flashing, unbrick, and IMEI backup toolkit**
built around **antumbra**.

This toolkit is intended for **advanced users**, developers, and device
recovery scenarios.

---

## Features

- Full RUI 5 unbrick / stock restore
- Safe IMEI backup and restore
- Windows and Linux support
- Deterministic flashing order
- No preloader flashing
- Service-grade scripts

---

## Supported Platforms

- **Windows** (Batch / CMD)
- **Linux** (Bash)
