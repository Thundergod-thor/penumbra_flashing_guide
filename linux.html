Linux Usage Guide

Instructions for using the Penumbra Toolkit on Linux.

1. Requirements

  • Native antumbra Linux binary
  • da.bin in bin/
  • udev permissions

No driver installation required.

2. Make Scripts Executable

chmod +x *.sh bin/antumbra

3. IMEI Backup (RECOMMENDED FIRST)

./imei-backup.sh

4. Full Unbrick / Flash

./unbrickscript.sh
  • A/B partitions flashed
  • super.img last
  • No preloader flashing

Run with sudo if device is not detected.

5. IMEI Restore (Optional)

./imei-restore.sh

Restore only from the same device.
